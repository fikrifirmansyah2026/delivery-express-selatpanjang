## 🚀 Cara Menggunakan

### 1. Clone Repository

```bash
git clone https://github.com/fikrifirmansyah2026/delivery-express-selatpanjang.git
cd delivery-express-selatpanjang
```

### 2. Buka di Browser

```
file:///path/to/index.html
```

### 3. Deploy ke GitHub Pages

- Push ke repository
- Aktifkan GitHub Pages di Settings
- Branch: `main`
- Folder: `/ (root)`
